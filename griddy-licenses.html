Additional Acknowledgments

Original Inspiration

Griddy is inspired by and incorporates code from Grids by Mutable Instruments.

The original Grids module uses drum patterns that were obtained through unsupervised learning from a large collection of drum loops. This project uses these same patterns and similar algorithms.
